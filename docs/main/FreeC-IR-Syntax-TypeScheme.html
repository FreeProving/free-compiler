Similar TypeScheme Source #

Two type schemes are similar if their abstracted types are similar under an extend Renaming that maps the corresponding type arguments to each other.

    Γ ∪ { α₁ ↦ β₁, …, αₙ ↦ βₙ } ⊢ τ ≈ τ'
