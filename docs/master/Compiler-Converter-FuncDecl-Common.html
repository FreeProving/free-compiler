Type inference

inferAndInsertTypeSigs :: DependencyComponent FuncDecl -> Converter (DependencyComponent FuncDecl) Source #

Infers the types of the function declarations in the given strongly connected component and inserts type signatures into the environment.

Returns the function declarations where the type arguments of all function and constructor applications on the right-hand side are applied visibly.
