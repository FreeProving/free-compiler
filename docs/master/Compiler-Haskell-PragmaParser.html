Compiler.Haskell.PragmaParser

Description

This module contains the parser for custom pragmas.

All custom pragmas have the format {--} (see customPragmaPattern).

The following custom pragmas are supported:

  • {--} annotates the decreasing argument of a function declared in the current module.
