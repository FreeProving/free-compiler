Identifies the decreasing arguments of the given mutually recursive function declarations.

Reports a fatal error message, if the decreasing arguments could not be identified.

Constant arguments

data ConstArg Source #

Data type that represents a constant argument shared by multiple mutually recusive functions.

Contains the names and indicies of the corrsponding arguments of the function declarations and a fresh identifier for the Variable sentence that replaces the constant argument.
