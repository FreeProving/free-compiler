Description

This module a state monad which allows the compiler's state (see Comiler.Environment) to be passed implicitly throught the converter.

There are also utility functions to modify the state and retreive information stored in the state.

State monad

type Converter = ConverterT Identity Source #

Type synonym for the state monad used by the converter.

All converter functions usually require the current Environment to perform the conversion. This monad allows these functions to pass the environment around implicitly.

Additionally the converter can report error messages and warnings to the user if there is a problem while converting.

Encapsulating environments

localEnv :: Monad m => ConverterT m a -> ConverterT m a Source #

Runs the given converter and returns its result but discards all modifications to the environment.

moduleEnv :: Monad m => ConverterT m (a, ModuleInterface) -> ConverterT m a Source #

Runs the given converter and returns its result.

Like localEnv, the modifications to the environment are discared but the returned ModuleInterface is inserted into the environment.

sectionEnv :: Monad m => ConverterT m a -> ConverterT m a Source #

Runs the given converter with envInSection set to True.

shadowVarPats :: Monad m => [VarPat] -> ConverterT m a -> ConverterT m a Source #

Adds entries for variable patterns during the execution of the given converter.

Unlike localEnv, all modifications to the environment are keept (except for added entries), except for the definition of the variables.
