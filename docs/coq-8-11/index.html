free-compiler-0.1.0.0: A Haskell to Coq compiler.

This compiler converts Haskell modules to Coq. Partial Haskell functions are modelled in Coq by applying a monadic transformation. We use the free monad for this purpose such that proofs about the converted Coq code can be monad/effect generic. This package is based on an early prototype developed at Flensburg University of Applied Sciences by Benedikt Jessen.
